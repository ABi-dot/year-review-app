# 个人年度总结 Web 应用

## 一句话描述
一个可以自动记录书/影/剧/游/地的个人年度消费轨迹工具，年底一键生成精美总结长图 + AI 文案。

## 为什么做这个项目
- 市面上没有完全匹配的产品（豆瓣是社区、Notion 模板太手动）
- 核心痛点：**记录太麻烦** + **总结太丑/没有** + **文案不会写**
- 闭环价值：自动化录入 → 精美可视化 → AI 文案 → 社交分享
