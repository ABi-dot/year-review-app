# 数据库设计草案

## 核心实体

### Item（条目）
一条记录对应一本书、一部电影、一款游戏或一个地点。

```prisma
model Item {
  id          String   @id @default(cuid())
  type        ItemType // BOOK, MOVIE, TV, GAME, PLACE
  title       String   // 名称
  originalTitle String? // 原名（外文作品）
  creator     String?  // 作者 / 导演 / 开发商 / 无
  coverUrl    String?  // 封面图 URL（R2 地址）
  externalId  String?  // 外部平台 ID（豆瓣 ID / Steam App ID 等）
  externalUrl String?  // 外部平台链接
  rating      Int?     // 评分 1-10
  review      String?  @db.Text // 短评
  finishedAt  DateTime? // 完成/体验日期
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt

  tags        Tag[]
  // 关联详情表
  bookDetail   BookDetail?
  movieDetail  MovieDetail?
  gameDetail   GameDetail?
  placeDetail  PlaceDetail?
}

enum ItemType {
  BOOK
  MOVIE
  TV
  GAME
  PLACE
}
```

### Tag（标签）
可自定义的标签，支持多对多关联。

```prisma
model Tag {
  id     String @id @default(cuid())
  name   String @unique // 如"力荐"、"治愈"、"烧脑"
  color  String? // 标签颜色，前端展示用
  items  Item[]
}
```

### 各类型详情表

```prisma
model BookDetail {
  id       String  @id @default(cuid())
  itemId   String  @unique
  item     Item    @relation(fields: [itemId], references: [id])
  isbn     String?
  publisher String?
  pageCount Int?
}

model MovieDetail {
  id        String  @id @default(cuid())
  itemId    String  @unique
  item      Item    @relation(fields: [itemId], references: [id])
  director  String?
  year      Int?
  duration  Int?    // 分钟
}

model GameDetail {
  id          String  @id @default(cuid())
  itemId      String  @unique
  item        Item    @relation(fields: [itemId], references: [id])
  platform    String? // PC / PS5 / Switch
  playtime    Int?    // 游玩时长（分钟）
  achievements Int?   // 成就数
}

model PlaceDetail {
  id       String  @id @default(cuid())
  itemId   String  @unique
  item     Item    @relation(fields: [itemId], references: [id])
  address  String?
  lat      Float?
  lng      Float?
  city     String?
}
```

### User Setting（用户设置）
```prisma
model UserSetting {
  id            String @id @default(cuid())
  steamId       String?
  doubanUserId  String?
  defaultAiStyle String @default("文艺") // 默认文案风格
}
```

### YearSummary（年度总结缓存）
生成长图后缓存，避免重复生成。

```prisma
model YearSummary {
  id        String   @id @default(cuid())
  year      Int
  template  String   // 模板类型
  imageUrl  String   // 生成的长图 URL
  content   String?  @db.Text // AI 生成的文案
  createdAt DateTime @default(now())
}
```

## 索引建议
- `Item(type, finishedAt)`：按类型+时间筛选最常用
- `Item(externalId, type)`：外部平台去重
- `Tag(name)`：标签搜索
